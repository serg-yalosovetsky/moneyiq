# User Stories

## Accounts

- As a user, I can create accounts with type, balance, currency, color, icon, and include-in-total behavior.
- As a user, I can see total balance across included accounts.
- As a user, I can edit or delete accounts.

## Categories

- As a user, I can see spending grouped by categories for the selected period.
- As a user, I can add, edit, archive, and delete categories.
- As a user, I can use root categories and subcategories.
- As a user, I can quickly add an expense/income from a category chip.

## Transactions

- As a user, I can add income, expense, transfer, borrow, lend, and repay transactions.
- As a user, I can search and filter operations.
- As a user, I can open category-specific transactions from category analytics.

## Budgets

- As a user, I can assign budget amounts to categories.
- As a user, I can inspect spending against budgets for the current period.

## Reports And Overview

- As a user, I can inspect aggregate finance summaries and category trends.

## Settings

- As a user, I can choose light/dark/system theme and an accent color.
- As a user, I can set the home screen default tab.
- As a user, I can show or hide the Budget tab.
- As a user, I can enable biometric login protection (triggers after 30s in background).
- As a user, I can enable daily spending notifications.
- As a user, I can set currency and number format.

## Data And Widgets

- As a user, I can use home-screen widgets for quick expense and balance visibility.
- As a user, I can use data import/export/backup surfaces where available.
